# Handleiding opmaken kaartapplicatie in nieuwe huisstijl

## 1. Template webmap
- **Intern:** [Template](https://gis.wageningen.nl/portal/home/item.html?id=c0d1021541044bb5a9ec000ed418904d)  
- **Extern:** [Template](https://wageningen.maps.arcgis.com/home/item.html?id=75f78d75b15c4bf5a9d1c1ef41064b1a)

**Stappen:**  
1. Open in Map Viewer  
2. Opslaan Als → naam: `WM_Onderwerp`  
3. In map **Kaartapplicaties** plaatsen  
4. Benodigde lagen toevoegen

## 2. Template webapp
- **Intern:** [Template](https://gis.wageningen.nl/portal/home/item.html?id=0b77fdc492bf4537ad1c9863aab10c47)  
- **Extern:** [Template](https://wageningen.maps.arcgis.com/home/item.html?id=d77c1c44d7c14b15b9e6e74f61f314c5)  
- **Voorbeeld:** [App](https://gis.wageningen.nl/portal/home/item.html?id=86e9945bb8434db5ad2785435562561b)

**Stappen:**  
1. Dashboard bewerken → Opslaan Als → naam: `WA_Onderwerp`  
2. In map **Kaartapplicaties** plaatsen  
3. Oude kaart verwijderen, nieuwe `WM_Onderwerp` toevoegen  

## 3. Opmaak & instellingen
**Overzichtskaart:**  
- Bijschrift Top: "Overzichtskaart" (**vet**, Kop 2, gecentreerd)  
- Tekstkleur: `#102652`  
- Voorgrondkleur: `#F5F5F5`  

**Legenda:**  
- Bijschrift Top: "Legenda" (**vet**, Kop 2, gecentreerd)  
- Zelfde kleuren als overzichtskaart  

**Layout:**  
- Verhouding: kaart 80%  
- Categorieselector instellen naar wens  

## 4. Notities
- Intern: titelstyling in zijbalk mogelijk  
- ArcGIS Online: titel opmaken via "Bijschrift Top"
